# 🏥 Health4You

**Health4You** is a **data analysis and visualization project** based on a fictional dataset of hospitals and patients across France.

The goal is to demonstrate how healthcare data can be structured, analyzed, and visualized using **Power BI**.

## 📊 Project Overview

This project analyzes:

* Hospital activity
* Patient demographics
* Hospital performance indicators
* Healthcare statistics in France

The results are presented in an **interactive Power BI dashboard**.

## 🗂 Dataset

The dataset is **synthetic and fictional**, created for data analysis purposes.

It includes:

* Hospital information
* Patient demographics
* Medical visits
* Treatments
* Regional distribution

## 📈 Dashboard Insights

The dashboard provides insights such as:

* Number of patients per hospital
* Regional healthcare distribution
* Age distribution of patients
* Most common treatments
* Hospital activity trends

## 🛠 Tools Used

* Power BI
* Data modeling
* Data visualization
